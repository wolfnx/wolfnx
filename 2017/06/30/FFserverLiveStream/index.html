使用FFserver搭建简单流媒体服务器

Posted on 2017-06-30 | Edited on 2020-03-02 | In Media | Views:

大概分为以下两个步骤

  1. 搭建一个流媒体服务器
  2. 验证流媒体服务器

搭建一个流媒体服务器

  1. 下载 ffserver

  2. 解压 ffserver.rar

  3. 复制视频到files中

  4. 编辑ffserver.conf

    <Stream httpTest.m3u8>
File "files/rabbit.avi"
Format avi
</Stream>

  5. 点击ffserver.bat（不要关闭此窗口）

  6. 在浏览器中打开locathost:8090/stat.html

验证流媒体服务器

  1. 使用ffplay播放流

  2. ip地址是本地网卡ip地址
